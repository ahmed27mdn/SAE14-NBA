# NBA Passion - Site Web (SAE 14)

Bienvenue sur le dépôt de mon projet de site web statique dédié à l'univers de la **NBA** et à ses légendes.
Ce projet a été réalisé dans le cadre de la **SAE 14** (Création d'un site Web) en 1ère année.

## Lien vers le site en ligne
Le site est hébergé via GitHub Pages et accessible à l'adresse suivante :
**[Visiter le site NBA Passion](https://ahmed27mdn.github.io/SAE14-NBA/)**

## Présentation du Projet
L'objectif de ce projet était de concevoir un site web complet, structuré et responsive en utilisant uniquement les langages standards du web, sans frameworks lourds (type Bootstrap).

Le thème choisi est la **National Basketball Association (NBA)**, un sujet passionnant permettant de mettre en avant :
- L'actualité et l'ambiance des matchs ("Le Show à l'Américaine").
- L'histoire de la ligue et ses joueurs mythiques (Jordan, LeBron, Kobe).
- Un formulaire de contact et de sondage pour la communauté.

### Objectifs Pédagogiques
- Maîtrise du **HTML5** sémantique.
- Maîtrise du **CSS3** (Flexbox, Responsive Design, Mise en forme de tableaux).
- Utilisation d'un gestionnaire de version (**Git/GitHub**).
- Respect des normes de base du web (W3C).

---

## Technologies Utilisées

Ce site a été développé avec une approche moderne et adaptative :

* **HTML** : Structure claire.
* **CSS** :
    * **Flexbox** pour la mise en page des cartes et du menu.
    * **Media Queries** pour le Responsive Design (Adaptation Mobile & Tablette).
    * **Transitions CSS** pour les effets de survol.
* **Git & GitHub** : Pour le versionnage et l'hébergement.
* **Google Fonts** : Polices *Oswald* (Titres) et *Roboto* (Texte courant).

---

## Fonctionnalités Clés

1.  **Design Responsive** : Le site s'adapte automatiquement :
    * Le menu devient vertical sur mobile.
    * Les cartes s'empilent pour une lecture facile sur petit écran.
2.  **Mise en page dynamique** :
    * Cartes interactives avec effet de zoom au survol.
    * Hero Banner (Bannière) avec image de fond immersive.
3.  **Contenu Riche** :
    * Tableau de statistiques avec gestion du défilement sur mobile.
    * Formulaire de contact stylisé avec menu déroulant (`<select>`).
4.  **Navigation Intuitive** : Barre de navigation fixe et liens clairs.

## Structure du Projet

L'architecture des fichiers est la suivante :

- **index.html** # Page d'accueil (Présentation générale & Cartes)
- **sujet.html** # Page Histoire (Chronologie & Tableau des stats)
- **apropos.html** # Page À Propos (Auteur & Formulaire de contact)
- **style.css** # Feuille de style unique gérant tout le design
- **README.md** # Documentation du projet
